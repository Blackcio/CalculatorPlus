This a simple calculator app, it provides more function than a just a four function calculator but it is not as complicated as a graphing calculator. It is intended for someone who needs a calculator but is looking for a little more than the basic Android calculator.

This program uses JavaCC, an open source parser generator and lexical analyzer generator, to parse mathematical phrases.

Features
* Parses mathematical phrases
* Stores history of input, and allows access to previous solutions
* Programmable variables
* Programmable functions
